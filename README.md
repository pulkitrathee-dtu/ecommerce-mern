# ShopSphere

🚧 **Status:** Under Development

## Overview

ShopSphere is a full-stack e-commerce web application built using the MERN stack. The project aims to provide a modern online shopping experience with secure authentication, product browsing, shopping cart functionality, and order management.

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Planned Features

### User
- User Registration & Login
- Browse Products
- Search & Filter Products
- Product Details
- Shopping Cart
- Checkout
- Order History

### Admin
- Add Products
- Edit Products
- Delete Products
- Manage Orders

## Project Structure

```
client/     # React frontend
server/     # Express backend
```

## Status

This project is currently under active development. New features and updates will be added regularly.
